## 🏠 Tamil Nadu House Prices – Full Stack ML Web App
A full-stack mini project built with React + FastAPI, predicting house prices across Tamil Nadu districts. Integrated with a PostgreSQL database (hosted on Neon Serverless) to store and showcase past user preferences and housing trends. Cold start handled via uptime pinger (Google Apps Script). The app delivers a complete end-to-end setup — frontend, backend, database, and deployment — all hosted seamlessly on Render.
